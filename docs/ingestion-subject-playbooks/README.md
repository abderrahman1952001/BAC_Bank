# Ingestion Subject Playbooks

Subject playbooks hold repeated, subject-specific ingestion lessons that are too
local for `.agents/skills/bac-ingestion/SKILL.md`.

Use them for:

- common hierarchy patterns
- polish conventions
- native-rendering defaults
- repeated extractor/import failure modes
- asset and crop conventions
- exemplar paper slugs or job ids when useful

Do not use them for one-paper quirks, universal ingestion workflow rules, or
instructions that conflict with the BAC ingestion skill and canonical workflow
docs.
